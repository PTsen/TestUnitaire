Pour utiliser l'application :

Il faut lancer l'application directement dans l'IDE (netbeans 8.0.2), car il faut utiliser le terminal.

Une fois l'application lancee, il y aura 2 fenetres qui va apparaitre.

Une fenetre avec le nom "Noble", qui sera l'observateur.

Une fenetre avec le nom, "add Observer".

la fenetre "Noble", peut s'ajouter dans la liste d'observateur de l'observable, en cliquant sur le l'onglet 
Option > Subscribe.

La fenetre "add Observer" permet tout d'ajouter des fenetre Noble en cliquant sur le bouton.

Pour voir la relation Observateur/Observable
il faut d'ecrire dans le terminal.
